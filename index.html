About CPUchain

CPUchain is an experimental digital currency that enables instant payments to anyone, anywhere in the world. CPUchain uses peer-to-peer technology to operate with no central authority: managing transactions and issuing money are carried out collectively by the network. CPUchain Core is the name of open source software which enables the use of this currency.

For more information, as well as an immediately useable, binary version of the CPUchain Core software, see https://github.com/cpuchain/cpuchain/releases.
